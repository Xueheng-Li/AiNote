# Workspace

Temporary workspace for AI-generated content, drafts, and work-in-progress files.

## Naming Conventions

- `temp_*` - Temporary files (auto-cleanup candidates)
- `draft_*` - Work in progress
- `script_*` - Generated scripts
- `pipeline_*` - Multi-step processing outputs
- `test_*` - Test files

## Maintenance

Review regularly and move important results to proper folders.
